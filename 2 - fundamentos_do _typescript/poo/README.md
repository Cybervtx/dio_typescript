# Poo - Fundamentos do TypeScript

Este é um projeto de estudo sobre Programação Orientada a Objetos (POO) utilizando TypeScript.

## Objetivos

- Entender os conceitos básicos de POO.
- Aprender a criar classes e objetos em TypeScript.
- Explorar os principais métodos e propriedades de classes.

## Tecnologias utilizadas

- TypeScript

## Como executar o projeto

1. Clone o repositório.
2. Instale as dependências com `npm install`.
3. Execute o projeto com `npm run dev`.
