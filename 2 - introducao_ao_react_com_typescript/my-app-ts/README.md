# My App — React + TypeScript

Uma aplicação de exemplo criada com Create React App + TypeScript, construída para estudos práticos e demonstração de UI com Chakra UI.

## Visão geral
Interface de login estilizada usando Chakra UI. Código focado em boas práticas com TypeScript e estrutura didática para aprendizado.

## Recursos
- Layout responsivo com Chakra UI
- Componentes tipados em TypeScript
- Estrutura simples para experimentos e exercícios

## Começando (rápido)
1. Instale dependências:
   ```sh
   npm install
   ```
2. Rode em modo de desenvolvimento:
   ```sh
   npm start
   ```
3. Abra http://localhost:3000

## Scripts úteis
- npm start — inicia o servidor de desenvolvimento  
- npm test — executa os testes  
- npm run build — gera build para produção

## Dependências principais
- React
- TypeScript
- Chakra UI

## Boas práticas
- Reinicie o TypeScript server no editor após instalar novas dependências (ex.: @chakra-ui/icons).
- Mantenha tipos explícitos e componentes pequenos e reutilizáveis.
